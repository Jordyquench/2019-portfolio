"Motion" animated text

A short looping motion graphic for the word "Motion"

This case study is a bit different from the others, I am going to use this page for completed and on going motion design/animation projects

The first motion graphic I am showing here is a short looping text animation, I was trying to get the hang of different aspects of within after effects and this was what came of it. It is quite a simple animation, it is basically shapes following a path into the frame and gradually revealling the text underneath, The Final part of the animation is the full stop bouncing off the O and this is done by following a path and tweaking with the speed graph to get the easing right.

I also did a version of the same animation but using the origin branding.

Dancing Flower

A 1930's style character animation

The idea for this came from a class I did on skillshare, I was wanting to try some character rigging and animation so I followed a class by Jake Bartlett which was focused on this. The class went throught everything from making sure the artwork was setup correctly within Illustrator, to the rigging process, to finally animationg a character within after effects.

To do the rigging in after effects I used A plugin called Duik Bassel, On first look it is very confusing, but for what I needed to do it wasn't to bad. I think with some more time I could get an even better hang of it, I would like to create some more characters to try and rig.

Squid Animation

Loosely in the 1930's style

I started this design kind of by mistake, I was drawing out some wireframes for a project in work and got distracted doodling, I ended up with this squid that looked like it could be a cool animation to try some new things and refine my skills in character rigging.

So I decided to Illustrate him up and animate him. I took a similar approach to the flower before but this time i wanted to rig the eyes up completely and give myself a bit more control with the limbs. I used Duik Bassel angain and can see some serious improvement in terms of my skill level with the plugin.

This time though I decided to go with some colour on the animation and ended up with two variations of the colour.

Can Animation

Faux 3D

The idea of faux 3D animation really intrigues me, So i decided to create a simple animation giving it a go. I seen a very simple tutorial oh how faux 3D works within after effects and this is the deisgn I came up with.

Originally I was going to go with a simple coca cola can but after I had the inital concept created I wanted to change it to something mpore unique.

The animation itself ended up being very simple to create. Once the animation was working the way I wanted it to, it was time to add some styling, so I added an inner shadow and a vignette, I think it really ads to the overall animation.

Monogram

Lopping animation

When I started creating this new portfolio website, I knew that I would need a new monogram, the one I had before wasn't somthing I liked anymore, or even the visual marque that I had.

Once the monogram was created I wanted to try animate it, nothing to crazy, but I knew I wanted it to loop, so that I could play it as a gif and it would just keep going.

I was very happy with the way the animation turned out, it is very simple but I think it does the job.
